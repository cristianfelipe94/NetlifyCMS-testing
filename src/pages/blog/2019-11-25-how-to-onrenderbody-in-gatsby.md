title: How to onRenderBody in Gatsby.
date: 2019-11-25T23:20:27.455Z
description: Render new metadata or new script tags in the building process.
---
Use the gatsby.ssr (gatsby Server Side Render) file to run APIs that are included into the file.

One of them is called onRenderBody, I used it to render script tags from Netlify CMS CDN.
